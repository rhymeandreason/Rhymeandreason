Code Editor

V.0.3

I made my own code editor. My goal is to make a simpler editor just for css, html, and markdown. This might be a bit of going down the rabbit hole of making tools to make tools, but there are some specific features I couldn't find elsewhere.

Mainly: view diffs highlighted inline. AI writes big chunks of code all at once, and I want to see what it did. Default is 'uncommitted changes' (which is what Zed gives you), but I also have 'last commit', because sometimes the agent commits all in one go.
